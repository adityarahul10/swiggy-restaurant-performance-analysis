Dataset Understanding

## Dataset Name: Swiggy Restaurant Dataset

## **Source:** [Kaggle - Swiggy Restaurant Dataset](https://www.kaggle.com/datasets/abhijitdahatonde/swiggy-restuarant-dataset/data)

## Description:

The dataset contains information about restaurants listed on Swiggy including restaurant names, locations, pricing, ratings, cuisine types, and delivery times.

# 📊 Dataset Information

This project uses the **Swiggy Restaurant Dataset**, which contains restaurant-level information collected from Swiggy across multiple Indian cities. The dataset provides insights into restaurant pricing, customer ratings, cuisine types, delivery times, and customer engagement, making it suitable for exploratory data analysis and business performance evaluation.

## 📋 Dataset Schema

| Column            | Data Type | Description                              |
| ----------------- | --------- | ---------------------------------------- |
| **ID**            | int       | Unique identifier for each restaurant    |
| **Area**          | nvarchar  | Locality where the restaurant is located |
| **City**          | nvarchar  | City in which the restaurant operates    |
| **Restaurant**    | nvarchar  | Restaurant name                          |
| **Price**         | float     | Average cost for two people (₹)          |
| **Avg_ratings**   | float     | Average customer rating                  |
| **Total_ratings** | smallint  | Total number of customer ratings         |
| **Food_type**     | nvarchar  | Primary cuisine served                   |
| **Address**       | nvarchar  | Restaurant address                       |
| **Delivery_time** | tinyint   | Estimated delivery time (minutes)        |
